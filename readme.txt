This plugin provides the most popular articles, using data collected by Wordpress.com Stats plugin.

== Description ==


Wordpress.com Popular Posts lists the most popular posts on a wordpress powered weblog.
This list can be used in the sidebar to provided visitors with an indication of what are the most visited pages.
